# Meetup Tracker App

A Meetup Tracker application built with React, Next.js, and MongoDB. The app allows users to view a list of meetups, view details of individual meetups, and create their own meetups.

## Features

- Displays a list of meetups on the home page.
- Users can click on individual meetups to view their details.
- Users can create their own meetups, providing relevant information.
- Utilizes MongoDB for data storage and retrieval.
- Connects to MongoDB using the `MongoClient.connect()` method.
- Styling is done using CSS Modules, enhancing component-based styling.
- Implements API routes for handling backend logic.
- Utilizes `next/router` for smooth navigation between pages.
- Implements dynamic routes for different meetup pages.

## Technologies Used

- React: JavaScript library for building user interfaces.
- Next.js: React framework for server-side rendering and routing.
- MongoDB: NoSQL database for storing and retrieving meetup data.
- `MongoClient.connect()`: Official MongoDB driver to connect to the database.
- CSS Modules: Styling approach using locally scoped class names.
- `next/router`: Next.js module for client-side routing.

## Getting Started

1. Clone this repository.
2. Install dependencies using `npm install`.
3. Set up a MongoDB database and configure your connection in the code.
4. Run the development server using `npm run dev`.
5. Visit `http://localhost:3000` in your browser to use the app.
